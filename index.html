jQuery

Плюсы
  • Просто работать с DOM
  • Есть библиотека базовых компонентов jQuery UI
  • Множество плагинов
  • Просто работать с Ajax
  • Широко распространен
Минусы
  • Недостаточно для современного SPA
Альтернатива
  • prototype.js
  • mootools

jQuery vs ...

Library, jQuery, prototype.js, mootools Usage, 70.2, 2.1, 3.5

ExtJS

Плюсы
  • Многое из коробки (MVVM, MVC, Store, etc.)
  • Широкая библиотека компонентов
  • Возможность писать свои компоненты
  • Быстрая разработка
Минусы
  • Сложная архитектура
  • Генерирует много HTML-кода
  • Дорогой
  • Отличное от стандартного окружение

React

Плюсы
  • Прост в освоении
  • Можно внедрять в отдельные части проекта
  • Компонентный подход к разработке
  • Много сторонних компонентов
Минусы
  • Библиотека, чего для SPA недостаточно

Интеграция приложений

  • Отсутствие единого подхода к разработке
  • Разные технологии в «подпроектах»
  • Выстраивание функциональности от пожеланий бизнеса
  • Отсутствие связей между приложениями

Монолит

Плюсы
  • Любые варианты взаимодействия между модулями/li>
  • Простой и понятный
Минусы
  • Нельзя обновлять по частям
  • Изменения в одном модуле могут затронуть другие

Использование iframe

Плюсы
  • Простое решение
  • Нет влияния изменений одного модуля на другие
Минусы
  • Слабая связанность отдельных приложений
  • Поддержка истории

F2

  • Загрузка в одну страницу контента с разных ресурсов
  • Взаимодействие между контейнером и преложениями и между приложениями внутри контейнера
